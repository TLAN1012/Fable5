CMini × Claude (Fable 5) 的公開作品集。

## 韓戰 3D 戰史劇場

電視歷史特別節目風格的互動式 3D 戰役地圖系列。每集都是單一 HTML 檔(Three.js,無建置流程):
3D 地形、敵我軍標(兵力+指揮官)、可拖動時間軸、事件字幕、晝夜循環、雪/砲火/爆破特效、
Web Audio 即時合成音效(無音檔,首次點擊頁面後啟用)、特寫運鏡、A−/A＋字體縮放。
