### 🎖 長津湖戰役與興南大撤退 1950 — 3D 戰史劇場

> **▶ 線上觀看:https://tlan1012.github.io/Fable5/chosin_1950/**

電視歷史特別節目風格的互動式 3D 戰役地圖,單一 HTML 檔(Three.js,無建置流程)。

- **時空範圍**:1950/11/27 – 12/24,蓋馬高原・長津湖・水門橋・興南港
- **敵我態勢**:藍=聯合國軍(美第10軍)、紅=志願軍第9兵團,各軍標附兵力與指揮官
- **時間軸**:可拖動/調速,13 個事件字幕——被圍、新興里覆滅、「換個方向進攻」突圍、
  水門橋三炸與 C-119 空投 M2 鋼橋、興南 193 艘船撤離 20 萬軍民、12/24 港口大爆破
- **特效**:零下暴風雪、晝夜循環(夜襲)、砲火、傘降動畫、港口連環爆破與衝擊波
- **音效**:Web Audio 即時合成(暴風雪、砲火、爆破、汽笛),無外部音檔;首次點擊頁面後啟用
- **操作**:滑鼠旋轉/縮放/平移、6 個特寫鏡頭、自動運鏡開關、A−/A＋ 字體縮放

> 兵力與傷亡數字採通行估計值,各方史料有差異,面板內已註明範圍。
